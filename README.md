#Salvar arquivos
- Salve arquivos de texto (extensão *.txt) dentro da pasta Notas
- Para acessar a pasta Notas basta clicar em "Abrir pasta" no CtrlCV Notes
- Pode criar subpastas para armazenas outras notas
- Os arquivos que não estão em subpastas serão exibidos na categoria "Favoritos"
- Se não for um arquivo de texto abrirá no app padrão para aquele arquivo

#Funcionamento
- Aperte Ctrl + Shift + Espaço para abrir a lista das notas
- Digite o nome da nota para filtrar
- Use as setas ↑ e ↓ para navegar entre as notas
- Aperte Enter ou clique 2 vezes com o mouse para escolher a nota
- Aperte Shift para escolher qual app vai abrir o arquivo
